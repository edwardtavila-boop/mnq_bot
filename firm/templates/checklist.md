# 60-Second Pre-Trade Checklist

Use for live trade decisions when time is short. Answer every item. If any cannot be answered, do not take the trade.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Instrument: _______  Direction: _______  Time: _______

[ ] SETUP — Name it exactly: _______
[ ] ENTRY — Exact price: _______
[ ] STOP — Exact price, structural reason: _______
[ ] TARGET — Exact price (min 3R): _______
[ ] R-MULTIPLE — Target÷Risk = _____ (kill if <3)
[ ] SESSION — Correct session for this setup? (Y/N)
[ ] REGIME — Current regime is one the strategy handles? (Y/N)
[ ] CATALYST — Scheduled event in next 30min? (Y→no trade)
[ ] STATE — Tilted, revenge-trading, over daily limit? (Y→no trade)
[ ] SIZE — Risk ≤ approved %? (Y/N)
[ ] KILL CRITERIA — What invalidates this trade? _______

GO / NO-GO: _______
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

If NO-GO, log the reason. Skipped trades are half your education — track them.
